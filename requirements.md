- __asyncio –__ Enables asynchronous, non-blocking network operations for high-performance concurrent port scanning.
- __argparse –__ Handles command-line arguments, allowing users to configure targets, ports, timeouts, and scan options.
- __socket –__ Provides low-level networking capabilities for creating TCP connections and supporting IPv4/IPv6 communication.
- __json –__ Used to serialize and export scan results in structured JSON format for automation and reporting.
- __csv –__ Allows exporting scan results into CSV files suitable for spreadsheets and security documentation.
- __datetime –__ Generates timestamps for each scanned port to support logging and audit trails.
- __typing –__ Improves code readability and reliability by adding type hints for data structures such as lists and dictionaries.
